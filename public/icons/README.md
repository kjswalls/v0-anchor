# Anchor PWA Icons

Drop the following files here (exported from Photoshop/Midjourney logo):

| File | Size | Purpose |
|------|------|---------|
| `icon-192.png` | 192×192 | Android/Chrome home screen (manifest) |
| `icon-512.png` | 512×512 | Android splash screen (manifest) |
| `icon-180.png` | 180×180 | iOS home screen → rename to `apple-touch-icon.png` in `/public/` |
| `icon-32.png`  | 32×32   | Browser favicon |
| `icon-16.png`  | 16×16   | Browser favicon (small) |
| `icon-1024.png`| 1024×1024 | Future App Store submission |

All files should be square PNGs with a transparent or solid background.
